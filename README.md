Code Robot Sumo - Lycée

Ce dépôt sert uniquement à stocker notre code pour faciliter le transfert entre la maison et les PC du lycée.

Où est le code ?

Le code Arduino principal se trouve dans le dossier Code_Arduino.
Le fichier à ouvrir avec l'IDE Arduino est Robot_Sumo.ino.

Pense-bête pour l'équipe

En début de séance (Maison ou Lycée) : Pensez bien à récupérer la dernière version du code en ligne avant de commencer à bosser (faire un Pull ou télécharger le zip).

En fin de séance : N'oubliez pas de sauvegarder votre travail sur GitHub (faire un Commit avec un petit message pour dire ce que vous avez changé, puis un Push) pour pouvoir le reprendre de l'autre côté !
